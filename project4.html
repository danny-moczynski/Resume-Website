This program allows users to play a spelling bee game is a web-based word puzzle that challenges players to create valid words from a set of letters, including a central letter.

  • Source code

Users can either play the games that are already made or they can add '/addPuzzle' to the url and add their own puzzles as well! This project provides an engaging and educational experience, encouraging players to expand their vocabulary, word recognition, and problem-solving skills
